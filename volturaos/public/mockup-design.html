Feature 1 — Quick-Add in Estimate Builder 3 LAYERS

A single "Quick Add" button replaces the category grid navigation. Tap it → bottom sheet with three modes: voice (default), instant search, recents. Works in both new estimates and change orders.

When a signed estimate needs more work added — outlet call turns into attic splice repair — tap "Add Change Order" on the job. Customer gets a second signature request covering only the new items. Original signature is preserved. Invoice includes everything.

What Gets Built

ComponentWhat it doesWhere it lives
NEWQuickAddSheet Bottom sheet with voice / search / recents tabs. Replaces category grid in estimate builder. components/estimate-builder/
NEWVoiceLineItems Mic button → Web Speech API transcript → Claude matches to pricebook → returns line items to confirm. components/estimate-builder/
NEWLineItemSearch Instant pricebook search with 2-letter debounce. Shows name, category, price, + Add button. components/estimate-builder/
NEWRecentsRow Pulls top 6 pricebook items by use_count. One-tap add. Auto-updates after every estimate saved. components/estimate-builder/
NEWChangeOrderBuilder Mini estimate builder pre-linked to a job + original estimate. Uses QuickAddSheet. Captures second signature. components/jobs/
NEWChangeOrderView Customer-facing page for signing. Shows original (greyed) + new items + total + signature canvas. app/change-orders/[id]/view/
UPDATEInvoiceDetail When job has change orders, invoice total = original estimate + all signed change orders. components/invoices/
UPDATEpricebook_items Add use_count + last_used_at. Increment use_count when item added to any estimate or change order. lib/actions/pricebook.ts
